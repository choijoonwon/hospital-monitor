# 현장 확인 질문 리스트
**내일 방문 전 확인해야 할 사항**

---

## 1. 모니터링 범위
- 현재 관리하는 병원이 총 몇 개인지? (우선순위 높은 병원 따로 있는지)
- 병원마다 키워드가 이미 정리되어 있는지, 아니면 새로 잡아야 하는지
- 네이버 카페만 모니터링하면 되는지, 블로그/플레이스도 필요한지
- 특정 카페만 보면 되는지 (성예사, 여우야 등 특정 카페 지정 여부)
- 바비톡, 강남언니처럼 네이버 외 플랫폼도 포함해야 하는지

## 2. 현재 업무 방식
- 지금 모니터링을 몇 명이 하는지, 각자 담당 병원이 나뉘어 있는지
- 현재 어떤 방식으로 결과를 보고하는지 (카카오톡, 엑셀, 구글시트 등)
- 부정글 발견 시 내부 보고 → 병원 보고까지 프로세스가 어떻게 되는지
- 하루에 몇 번 정도 모니터링하는지 (아침 1회인지, 수시로인지)

## 3. 컴퓨터 환경
- 이 프로그램을 몇 대 컴퓨터에 설치해야 하는지
- 각 컴퓨터에 Chrome이 설치되어 있는지
- 인터넷은 유선인지 와이파이인지 (속도 확인)
- 회사 컴퓨터에 프로그램 설치 권한이 있는지 (보안 정책)

## 4. 네이버 계정
- 네이버 API 키 발급할 계정이 있는지 (법인 or 개인)
- 플레이스/카페 수집에 네이버 로그인이 필요한 경우 어떤 계정 쓸 건지

## 5. 추가 기능 필요 여부
- 수집 결과를 엑셀로 내보내기 기능이 필요한지
- 부정글 발견 시 카카오톡 자동 알림이 필요한지
- 병원별로 담당자가 달라서 권한 분리가 필요한지
- 수집 이력을 얼마나 오래 보관할지
