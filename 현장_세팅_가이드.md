# 현장 세팅 가이드
**[엠에이디씨] 병원 부정글 모니터링 시스템**

---

## 시작 전 준비물 체크
- [ ] 인터넷 연결된 Windows 컴퓨터
- [ ] Chrome 브라우저 (설치 안 되어 있으면 먼저 설치)
- [ ] 네이버 계정 1개 (API 키 발급용)

---

## STEP 1 — 파일 다운로드

1. Chrome 브라우저를 열고 주소창에 입력:
   ```
   https://github.com/choijoonwon/hospital-monitor
   ```

2. 초록색 **`<> Code`** 버튼 클릭

3. **`Download ZIP`** 클릭 → 다운로드 폴더에 저장됨

4. 다운로드된 `hospital-monitor-master.zip` 파일 **우클릭 → 압축 풀기**
   - 압축을 C드라이브 바로 아래에 풀기 권장:
     `C:\hospital-monitor\`

---

## STEP 2 — Python 설치

> 이미 설치되어 있으면 STEP 3으로 건너뛰세요.
> 확인 방법: 키보드 `윈도우키 + R` → `cmd` 입력 후 엔터 → 검은창에 `python --version` 입력 → 버전 숫자가 나오면 설치된 것

1. Chrome에서 접속:
   ```
   https://www.python.org/downloads/
   ```

2. 노란색 **`Download Python 3.xx.x`** 버튼 클릭

3. 다운로드된 파일 실행 후 **반드시** 아래 체크박스 선택:
   > ✅ **Add Python to PATH** (이거 안 하면 나중에 오류남!)

4. **Install Now** 클릭 → 설치 완료 대기

---

## STEP 3 — 초기 설치 (최초 1회만)

1. 압축 푼 폴더(`C:\hospital-monitor\`) 열기

2. **`setup.bat`** 파일 더블클릭

3. 검은 창이 뜨면서 자동 설치 진행 (1~3분 소요)

4. **`설치 완료! run.bat 을 실행하면 시작됩니다`** 메시지가 나오면 완료

   > ⚠️ 오류가 뜨는 경우:
   > - `Python을 찾을 수 없습니다` → STEP 2로 돌아가서 Python 재설치 (PATH 체크 확인)
   > - `인터넷 연결을 확인해주세요` → 와이파이/인터넷 연결 후 다시 실행

---

## STEP 4 — 네이버 API 키 발급 (5분)

> 병원 키워드 검색에 사용하는 무료 키입니다. 하루 25,000건 무료.

1. Chrome에서 네이버 로그인 후 접속:
   ```
   https://developers.naver.com/apps/#/register
   ```

2. **애플리케이션 이름**: 아무거나 입력 (예: `모니터링`)

3. **사용 API**: 스크롤 내려서 **`검색`** 체크

4. **WEB 서비스 URL**: `http://localhost` 입력

5. **등록하기** 버튼 클릭

6. 등록 완료 페이지에서 **`Client ID`** 와 **`Client Secret`** 복사해두기
   (Client Secret은 눈 아이콘 클릭해야 보임)

---

## STEP 5 — 병원 정보 입력

**방법 A: 대시보드에서 직접 입력 (권장)**

1. `run.bat` 더블클릭 → 브라우저 자동 오픈
2. 상단 **`⚙ 병원 관리`** 버튼 클릭
3. **`+ 병원 추가`** 클릭 → 병원명 입력
4. 키워드 입력칸에 검색어 추가 (예: `리팅성형외과`, `ㄹㅌ`, `000원장`)
5. **저장** 클릭

**방법 B: 파일 직접 수정**

1. `C:\hospital-monitor\config.json` 파일 **우클릭 → 메모장으로 열기**

2. 아래 형식으로 수정:
   ```json
   {
     "naver_client_id": "여기에_Client_ID_붙여넣기",
     "naver_client_secret": "여기에_Client_Secret_붙여넣기",
     "hospitals": [
       {
         "name": "리팅성형외과",
         "keywords": ["리팅성형외과", "ㄹㅌ", "리팅", "000원장"]
       },
       {
         "name": "강남피부과",
         "keywords": ["강남피부과", "ㄱㄴ피부"]
       }
     ]
   }
   ```

3. **파일 → 저장** (Ctrl+S)

   > ⚠️ 주의: 쉼표(`,`), 따옴표(`"`) 빠지거나 추가되면 오류남. 형식 그대로 유지!

---

## STEP 6 — 실행

1. **`run.bat`** 더블클릭

2. 검은 창이 뜨고 브라우저에서 대시보드 자동 오픈

3. 대시보드에서 **`▶ 지금 수집하기`** 클릭 → 수집 시작

4. 완료 후 테이블에 결과 표시됨

   > ⚠️ API 키 오류 메시지가 뜨는 경우:
   > `config.json`에 Client ID / Secret이 정확히 입력됐는지 확인

---

## 일상적인 사용법

| 상황 | 방법 |
|------|------|
| 매일 아침 모니터링 | `run.bat` 실행 → `▶ 지금 수집하기` 클릭 |
| 새 병원 추가 | `⚙ 병원 관리` → `+ 병원 추가` |
| 병원 키워드 수정 | `⚙ 병원 관리` → 해당 병원 키워드 수정 후 저장 |
| 부정글 보고 | 해당 글 우측 `복사` 버튼 → 카카오톡에 붙여넣기 |
| 확인한 글 정리 | `✔ 전체 확인 처리` 클릭 → NEW 배지 제거 |
| 신규 글만 보기 | `신규만 보기` 체크 → `🔍 적용` |

## 자주 묻는 오류

**Q. 브라우저가 안 열려요**
→ `run.bat` 실행 후 Chrome 주소창에 직접 `http://localhost:8080` 입력

**Q. "localhost에 대한 액세스가 거부됨" 오류**
→ `run.bat`이 실행 중인지 확인. 검은 창이 떠 있어야 함

**Q. 수집하기 눌렀는데 결과가 안 나와요**
→ `config.json`의 API 키 확인. 네이버 개발자 센터에서 키 재발급 후 다시 입력

**Q. 병원 추가했는데 드롭다운에 안 보여요**
→ `⚙ 병원 관리`에서 저장 눌렀는지 확인 후 페이지 새로고침(F5)

**Q. 파일을 수정했는데 반영이 안 돼요**
→ `run.bat` 창 닫고 다시 실행
